# AI QA Comparison Project

## Overview
This project evaluates AI tools (ChatGPT vs CodeGPT) for generating API test cases and analyzing workflow usability in QA testing scenarios.

## Tools Evaluated
- ChatGPT
- CodeGPT (VS Code extension)

## Focus Areas
- API Test Case Generation
- Positive, Negative, Edge, and Security Test Design
- Workflow Usability
- Freemium Tool Limitations

## Key Finding
ChatGPT provided complete structured QA outputs, while CodeGPT was limited due to freemium paywall restrictions blocking chat functionality.

## Skills Demonstrated
- QA Test Case Design
- AI Tool Evaluation
- Exploratory Testing
- Defect Reporting
- Technical Documentation (Markdown)
